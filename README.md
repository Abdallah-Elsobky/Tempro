# Tempro 🌤️

Tempro is a modern, beautifully designed iOS weather application built entirely with SwiftUI. It provides users with accurate real-time weather data, hourly and daily forecasts, and unique lifestyle indicators tailored to current weather conditions.

## ✨ Features

- **Current Weather**: Instantly view the weather conditions for your current location upon opening the app.
- **Hourly & Daily Forecasts**: Get a detailed breakdown of the weather for the next few days and hours.
- **Global Search & Favorites**: Search for any city worldwide and add it to your saved locations for quick access.
- **Lifestyle Indicators**: Smart recommendations for outdoor activities like running, hiking, fishing, and golfing based on real-time weather metrics (temperature, humidity, visibility, and precipitation).
- **Premium UI/UX**: Designed with a sleek, native SwiftUI interface that supports both light and dark modes, featuring smooth animations and a premium look.

## 🛠 Tech Stack

- **SwiftUI**: For a declarative, native, and responsive user interface.
- **MVVM Architecture**: Ensures a clean separation of concerns, making the codebase scalable and easy to test.
- **CoreLocation**: Handles seamless and fast user location retrieval.
- **Combine**: Used alongside `@Published` properties for reactive state management.
- **Alamofire**: Robust and elegant networking layer to fetch data from the weather API.

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abdallah-Elsobky/Tempro.git
   cd Tempro
   ```
2. **Open the project in Xcode:**
   Open `Tempro.xcodeproj` or double-click it in Finder.
3. **Install Dependencies:**
   The project uses Swift Package Manager (SPM). Xcode will automatically resolve and fetch dependencies like Alamofire when you open the project.
4. **Build and Run:**
   Select your preferred iOS Simulator or physical device and press `Cmd + R` (or click the Run button).
